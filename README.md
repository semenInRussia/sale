Моя домашка("Подписки и скидки") по предмету ООП в СПбГУ МКН первый семестр направления Современное Программирование.

Задача домашки была преобразовать код с использованием функциональных возможностей языка C#

Точно были выполнены требования из [OOPDay1.1.html](./FuncWithOOP.html)

Классы реализованы в файлах `Book.cs`, `BookStore.cs` и `Catalog.cs` (каждый класс в своём файле).  В `Program.cs` написан пример использования данных классов.

Для удобства использования классы располагаются в namespace `Bookstore`.

Для запуска используйте:

```shell
dotnet run
```
